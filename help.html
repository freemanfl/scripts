Information and Help for the extension

How to install

  1. Open the link to Chrome Store
  2. Click "Install"
  3. Do not forget to pin the extension to the toolbar
  4. Click extension icon on any Ford page

How to use

  1. Your current location in bold
  2. Destinations you can go to
  3. Destination URL
  4. Settings button
  5. Copy destination URL
  6. Copy destination URL starting from the '/content...' part
  7. Information and Help button

What can go wrong

  • Requires this component in order to go from the editing of BETA pages to perf/live. Sometimes this component fails to load and you will have to reload the page.

    But also, DSE pages don't have this component, hence the jump from editing to perf/live for these page WILL NOT WORK.

  • Sometimes it may look like you are in the Classic UI, but you actually are not. When you open links from the worfklow, it opens the editor, but the URL is missing "CF#"
